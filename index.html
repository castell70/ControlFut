              Ayuda — ControlFut

              Introducción

              ControlFut es una aplicación ligera para gestionar jugadores, personal y sesiones de entrenamiento. Su objetivo es centralizar datos del equipo y facilitar el análisis y el seguimiento.

              Opciones principales

              • Dashboard: Resumen de estadísticas y jugadores recientes.
              • Registrar Jugador: Formulario para añadir o editar jugadores y listado con filtros.
              • Registrar Administrativo: Añadir personal administrativo del club.
              • Registrar Soporte: Añadir personal de soporte (fisioterapeutas, médicos).
              • Gestionar Entrenamientos: Crear sesiones, registrar asistencia y ver lista de sesiones.
              • Análisis de Jugadores: Filtrar jugadores por posición/estado/rendimiento y ver perfil detallado.
              • Herramientas: Exportar datos a JSON e importar JSON para restaurar o transferir información.

              Ejemplos de uso

              • Registrar un jugador: Ve a "Registrar Jugador", completa datos (nombre, número, posición) y presiona "Guardar jugador".
              • Marcar asistencia en una sesión: En "Gestionar Entrenamientos" crea una sesión y coloca IDs de jugadores separados por comas (por ejemplo "p1,p3") para registrar asistencia.
              • Exportar datos: Abre "Herramientas" y selecciona "Exportar Datos (JSON)" para descargar un archivo con toda la base de datos.
              • Ver perfil y agregar nota: Desde cualquier listado presiona "Ver" en un jugador, luego "Agregar nota" para guardar observaciones de seguimiento.
